A library for building autonomous agents that reason ethically

Build an autonomous agent

Define how your agent interacts with the world by specifying its states, actions, transition function, and reward function. Select the classic grid world agent or make your own.

const agent = new Agent(...)

Choose an ethical system

Give your agent the context needed to reason about the ethical implications of its actions. Select an ethical system that aligns with important values, risks, and responsibilities.

const ethics = new Ethics(...)

Bring them together

Generate a policy for your agent that allows it to complete tasks as efficiently as possible while still complying with an ethical system. Say goodbye to reward hacking.

const solution = morality.solve(...)

Getting Started

Morality.js is a library for building autonomous agents that reason ethically. To get started, let's go through a simple example on how to build a grid world agent that follows an ethical system. However, before we get started, you can add Morality.js to your node project like any other npm package like this:


  npm install --save morality

Suppose we have a robot searching for money in a grid world. The robot can stay in its current location or move to another square by going north, east, south, or west. When the robot tries to move to another square, it has a small chance of slipping into a square to the left or right of its current square because one of its wheels can get stuck sometimes. In Morality.js, we can define a grid world agent using a multidimensional array where the symbol O represents an empty square, the symbol W represents a wall square, and the symbol G represents a goal square.


  const agent = new GridWorldAgent({
    "width": 5,
    "height": 3,
    "grid": [
      ['O', 'O', 'W', 'W', 'O'],
      ['O', 'O', 'W', 'W', 'O'],
      ['O', 'O', 'O', 'O', 'G']
    ]
  });

If we want our robot to reason ethically, we also have to specify an ethical system. Morality.js offers three ethical systems right out of the box: forbidden state ethics, prima facie duty ethics, and moral exemplar ethics. Forbidden state ethics is a simple ethical system that says that an agent shouldn't ever enter certain states of the world. In Morality.js, we can define forbidden state ethics by listing each forbidden state.


  const ethics = new ForbiddenStateEthics([0, 4, 10]);

Prima Facie Duty Ethics is a more sophisticated ethical system that says that an agent should follow duties. To do this, we need a few components: a set of duties, a violation function, a penalty function, and a tolerance. First, the set of duties represent the principles that shouldn't be neglected by the agent. Next, the violation function represents whether or not each state neglects a duty. Moroever, the penalty function represents the cost of neglecting a duty given that the agent performed a specific action in a particular state. Finally, the tolerance represents the slack that is given to the agent. In Morality.js, we can define prima-facie duty ethics by specifying each of these components.


  const duties = ['Quiet Operation', 'Personal Space'];

  const violationFunction = (state) => {
    if (state == 55) {
      return ['Quiet Operation', 'Personal Space'];
    }
    return [];
  };

  const penaltyFunction = (duty, state, action) => {
    if (duty == 'Quiet Operation') {
      return 1;
    }
    if (duty == 'Personal Space') {
      return 10;
    }
    return 0;
  };

  const tolerance = 0.3;

  const ethics = new PrimaFacieDutyEthics(duties, violationFunction, penaltyFunction, tolerance);

Moral exemplar ethics is an ethical system based on virtue ethics that says that an agent should strive to act like a moral exemplar. To put matters more simply, an agent should perform any action that would be performed by a moral exemplar. This means we need a list of trajectories that have been performed by moral exemplars. In Morality.js, we can define moral exemplar ethics by specifying this list.


  const ethics = new MoralExemplarEthics([
    [[0, 1], ['EAST', 'SOUTH']], 
    [[5, 6], ['SOUTH'], ['EAST']]
  ]);

Once we've defined an autonomous agent and an ethical system, we can use Morality.js to calculate a solution.


  const solution = morality.solve(agent, ethics);

A solution in Morality.js is just a map with a value function linking each state to its optimal value and a policy linking each state to its optimal action.


  {
    'values': {
      '0': 84.9,
      '1': 85.5,
      ...
    },
    'policy': {
      '0': 'SOUTH',
      '1': 'EAST', 
      ...
    }
  }
