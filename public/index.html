A library for building autonomous agents that reason ethically

Build an agent

Define how your agent interacts with the world by specifying its states, actions, transition function, and reward function. Select the classic grid world agent or make your own.

const agent = new GridWorldAgent(...)

Choose an ethical theory

Give your agent the context needed to reason about the ethical implications of its actions. Select an ethical theory that aligns with important values, risks, and responsibilities.

const ethics = new ForbiddenStateEthics(...)

Bring them together

Generate a policy for your agent that allows it to complete tasks as efficiently as possible while still complying with the given ethical theory. Say goodbye to reward hacking.

const solution = morality.solve(...)

Playground

Suppose we have a robot searching for money hidden in a grid world. The robot can stay in its current location or move to another square by going north, east, south, or west. When the robot moves to another square, it has a small chance of slipping into a square to the left or right of its current square because one of its wheels can get stuck sometimes. More importantly, because we want our robot to be moral, we have to specify an ethical framework. While there are many ethical frameworks out there, we use two simple choices here: forbidden state ethics and norm-based ethics. Simply put, in forbidden state ethics, the robot has to avoid forbidden squares at all costs. In norm-based ethics, however, the robot has to avoid squares that break specific rules but is allowed some slack. The goal of the robot is to get as much money as possible while staying moral.

Help

Click the Clear button to reset the grid world and start over.

Select from the dropdown on the left to change whether to see the action or value of each square.

Select from the dropdown on the right to change the ethical context.

Click on a square and select from its popover menu to edit the grid world.

Each square shows the amoral policy in blue and the moral policy in green. They're outlined in red when they disagree.

The green part of the top-right bar is the value of the moral policy.

The red part of the top-right bar is the price of morality, which is the difference in value between the amoral policy and the moral policy.
